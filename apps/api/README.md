# Basmati (NestJS Backend)

Basmati is the core engine of the Pulao project. It handles all business logic, database interactions, and real-time communication.

## Tech Stack
- **NestJS**: A progressive Node.js framework.
- **TypeScript**: For type-safe development.
- **Socket.io**: (Upcoming) For real-time "Pulse" communication.

## Getting Started
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Run in development mode:
   ```bash
   pnpm start:dev
   ```

## Folder Structure
- `src/main.ts`: Application entry point.
- `src/app.module.ts`: Root module.
