# Saffron (Next.js Web)

Saffron provides the premium web dashboard for the Pulao project.

## Tech Stack
- **Next.js**: React framework for production.
- **Tailwind CSS**: For styling.
- **TypeScript**: Shared types with the backend.

## Getting Started
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Run the development server:
   ```bash
   pnpm dev
   ```

## Integration
Saffron is designed to consume data from **Basmati** and share types with **Cilantro** via the **Masala** package.
