# Cilantro (Expo Mobile)

Cilantro is the fresh mobile interface for the Pulao project, built with React Native and Expo.

## Tech Stack
- **Expo**: Accelerated React Native development.
- **React Native**: Native UI components.
- **TypeScript**: Unified type-safety.

## Getting Started
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Start the development server:
   ```bash
   pnpm start
   ```
3. Open in **Expo Go** on your physical device or use a simulator.

## Key Features
- Clean, dark-themed UI.
- (Upcoming) Real-time notifications from Basmati.
